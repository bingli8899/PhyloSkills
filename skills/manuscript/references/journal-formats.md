# Journal Format Guide — Plant Phylogenetics

Target journals for plant systematic and phylogenetic studies.
Use when formatting the manuscript in Step 2 of the `manuscript` skill.

---

## Taxon

**Publisher:** Wiley / International Association for Plant Taxonomy  
**Scope:** Plant taxonomy, systematics, nomenclature, evolutionary biology  
**Impact factor:** ~3.0  
**Open access:** Optional (hybrid)  
**URL:** https://onlinelibrary.wiley.com/journal/19968175

| Section | Requirements |
|---------|-------------|
| Citation style | Author-year (Harvard): (Smith & Jones, 2020) |
| Reference format | Smith AB, Jones CD. 2020. Title of paper. *Taxon* 69: 123–145. |
| Methods length | No explicit limit; typically 600–1,200 words |
| Figures | 300 dpi min; TIFF or EPS preferred; max 85 mm (1-col) or 175 mm (2-col) |
| Supplementary | Accepted; upload separately |
| GenBank requirement | Accession numbers required in text or supplementary table |
| Tree deposition | TreeBASE or Dryad recommended but not always mandatory |
| Subheadings in Methods | "Taxon sampling," "DNA extraction," "PCR amplification," "Sequence alignment," "Phylogenetic analyses" (adapt as needed) |
| Word limit | ~8,000 words (excluding references and captions) |

**Key Methods section conventions for Taxon:**
- Always state GenBank accession numbers in a supplementary table
- Use "Sequences were aligned" not "We aligned sequences" (passive voice preferred)
- Report bootstrap as "BS" and posterior probability as "PP"
- Cite software with version number in parentheses after first mention

## Systematic Botany

**Publisher:** American Society of Plant Taxonomists  
**Scope:** Plant systematics, evolution, biogeography  
**Impact factor:** ~2.5  
**Open access:** No (subscription); select open access available  
**URL:** https://www.jstor.org/journal/systematicbotany

| Section | Requirements |
|---------|-------------|
| Citation style | Author-year: (Smith and Jones 2020) — note: no comma before year |
| Reference format | Smith A. B. and C. D. Jones. 2020. Title. *Systematic Botany* 45: 123–145. |
| Methods length | Concise; typically 500–1,000 words |
| Figures | 300 dpi; TIFF; 83 mm (1-col) or 174 mm (2-col) wide |
| Supplementary | Appendices published in journal; supplementary online also accepted |
| GenBank requirement | All sequences must be deposited; accessions in appendix table |
| Tree deposition | TreeBASE submission required for published matrices |
| Methods subheadings | "Plant Material," "DNA Extraction and PCR Amplification," "Phylogenetic Analyses" |

**Key conventions:**
- Submit alignment matrices and trees to TreeBASE before submission
- Voucher specimens must be cited for each accession
- "Systematic Botany" uses "and" not "&" in in-text citations

## American Journal of Botany

**Publisher:** Wiley / Botanical Society of America  
**Scope:** Plant biology broadly; systematics, evolution, ecology  
**Impact factor:** ~3.5  
**Open access:** Optional (hybrid)  
**URL:** https://bsapubs.onlinelibrary.wiley.com/journal/15372197

| Section | Requirements |
|---------|-------------|
| Citation style | Author-year: (Smith and Jones, 2020) |
| Reference format | Smith, A. B., and C. D. Jones. 2020. Title. *American Journal of Botany* 107: 1–15. |
| Methods length | No limit; typically 800–1,500 words |
| Figures | 300 dpi min; TIFF/EPS/PDF; 88 mm (1-col) or 180 mm (2-col) |
| Supplementary | Accepted; free data deposition in Dryad |
| GenBank requirement | Required; include in supplementary Data S1 |
| Tree deposition | Dryad (preferred) or TreeBASE |
| Methods subheadings | "Taxon Sampling," "Molecular Methods," "Sequence Alignment," "Phylogenetic Analysis" |

**Key conventions:**
- Separate "Molecular Methods" (lab) from "Phylogenetic Analysis" (computational)
- First use of a statistical method should cite the primary reference
- AJB requires Data Availability Statement

## Molecular Phylogenetics and Evolution

**Publisher:** Elsevier  
**Scope:** Molecular evolutionary biology; phylogenetics; population genetics  
**Impact factor:** ~4.0  
**Open access:** Optional (hybrid); Gold OA available  
**URL:** https://www.sciencedirect.com/journal/molecular-phylogenetics-and-evolution

| Section | Requirements |
|---------|-------------|
| Citation style | Numbered: [1] or (Smith et al., 2020) depending on section |
| Reference format | Smith, A.B., Jones, C.D., 2020. Title. *Mol. Phylogenet. Evol.* 150, 106876. |
| Methods length | No limit; typically 1,000–2,000 words |
| Figures | 300 dpi (halftone) or 1,000 dpi (line art); TIFF; 90 mm (1-col) or 190 mm (2-col) |
| Supplementary | Accepted freely |
| GenBank requirement | Required; accession list as supplementary table |
| Tree deposition | TreeBASE or Dryad; include Newick trees as supplementary |
| Methods subheadings | "Taxon Sampling and Sequence Data," "Sequence Alignment," "Phylogenetic Analyses," "Divergence Time Estimation" |

**Key conventions:**
- "et al." after 2 authors in in-text citations
- Separate section for divergence time estimation if BEAST2 used
- Include convergence diagnostics (ASDSF, ESS) explicitly in text or table

## Systematic Biology

**Publisher:** Oxford University Press / Society of Systematic Biologists  
**Scope:** Theoretical and methodological phylogenetics; highest-impact systematics  
**Impact factor:** ~14  
**Open access:** Optional; Advance Access publication  
**URL:** https://academic.oup.com/sysbio

| Section | Requirements |
|---------|-------------|
| Citation style | Author-year: (Smith and Jones 2020) |
| Reference format | Smith A.B., Jones C.D. 2020. Title. *Syst. Biol.* 69:123–145. |
| Methods length | Concise; reviewers expect precision over completeness |
| Figures | 300 dpi; EPS or TIFF; single column (88 mm) or double (180 mm) |
| Supplementary | Free; often extensive for methods-heavy papers |
| GenBank requirement | Mandatory; online table |
| Tree deposition | TreeBASE or Dryad mandatory; include all trees and alignments |
| Methods subheadings | "Data," "Analyses" (minimal subheadings preferred) |

**Key conventions:**
- Highly competitive; methods must be rigorous and clearly justified
- Avoid redundancy — cite the primary method paper, not every use
- All analyses must be reproducible; code/scripts should be deposited
- Concordance factors and gene tree discordance expected for multi-locus studies

## PLOS ONE

**Publisher:** PLOS  
**Scope:** Broad; any sound science  
**Impact factor:** ~3.5  
**Open access:** Full OA (APC required)  
**URL:** https://journals.plos.org/plosone

| Section | Requirements |
|---------|-------------|
| Citation style | Numbered superscript: Smith and Jones¹ |
| Reference format | Smith AB, Jones CD (2020) Title. *PLOS ONE* 15(3): e0123456. |
| Methods length | No limit; must be complete and reproducible |
| Figures | 300 dpi; TIFF/EPS; max 6.83 in (1-col) |
| Supplementary | Encouraged; S1 File, S1 Table, etc. |
| GenBank requirement | Required; PLOS will check accession availability |
| Tree deposition | Not mandatory but recommended; Dryad integration available |
| Data Availability | Mandatory statement |

## Data Deposition Checklist

Before manuscript submission, complete all deposition steps:

- [ ] GenBank: all sequences with consistent accession prefix, linked to the study
- [ ] TreeBASE (https://treebase.org): upload alignment matrix and tree(s) — get Study ID
- [ ] Dryad (https://datadryad.org): large datasets, scripts, supplementary data — get DOI
- [ ] GitHub (optional): analysis scripts; include version tag / release for the submission version

**Data Availability Statement template:**
> "All sequences generated in this study are deposited in GenBank (accessions [LIST]).
> The alignment matrix and phylogenetic trees are available in TreeBASE (study [ID]).
> Analysis scripts are available at [GitHub URL] (version [TAG])."
